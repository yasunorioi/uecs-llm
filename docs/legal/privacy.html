プライバシーポリシー

最終更新日: 2026年3月13日

AgriHA LINE Bot（以下「本サービス」）は、北海道の施設園芸（ビニールハウス）向け環境制御支援サービスです。本ポリシーは、本サービスにおける個人情報の取り扱いについて定めます。

1. 取得する情報

  • LINE プラットフォームから提供されるユーザー ID（LINE User ID）
  • 本サービスとの対話内容（テキストメッセージ）
  • 温室センサーデータ（気温、湿度、日射量、CO2濃度等）

2. 利用目的

  • 温室環境の監視・制御に関する情報提供
  • ユーザーからの指示に基づくリレー制御の実行
  • 異常値検知時のアラート通知
  • サービスの改善・不具合対応

3. 第三者提供

取得した情報を第三者に提供することはありません。ただし、以下の場合を除きます。

  • ユーザーの同意がある場合
  • 法令に基づく場合
  • LLM API プロバイダー（Anthropic 等）への対話内容の送信（制御判断のため）

4. データの保存

  • 対話履歴: ローカルデータベースに保存（Raspberry Pi 上）
  • センサーデータ: ローカルデータベースに保存
  • 外部クラウドへのデータ蓄積は、ユーザーが明示的に設定した場合のみ行われます

5. データの削除

ユーザーは LINE Bot をブロックすることで、以降のデータ取得を停止できます。保存済みデータの削除を希望する場合は、管理者にご連絡ください。

6. お問い合わせ

本ポリシーに関するお問い合わせは、GitHub リポジトリの Issue またはサービス管理者へご連絡ください。
